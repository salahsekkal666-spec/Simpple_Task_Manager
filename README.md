# Simple Task Manager

Mini project for PhD Programming course – University of Mohamed Kheider – Biskra  
Academic Year: 2025/2026  
Author: Salah Sekkal

---

## Description

This is a simple command-line Task Manager written in Python.  

It allows students to manage their study/homework tasks easily.  
The program can:

- Add new tasks
- View all tasks
- Mark tasks as done
- Save tasks using JSON
- Load tasks when the program starts

This project demonstrates:

- File handling in Python  
- Using JSON for data storage  
- Writing clean, modular code using functions  
- Handling user input errors  
- Working with Git and GitHub
